# Vue.js + vue-router sample application.
Vue.js + vue-router + Elementを使用したサンプルアプリケーション。
Vue.jsはWebpackでビルドし、サーバはWebpack-dev-serverを利用する。

## インストール
1. `npm install`
2. `npm start`
