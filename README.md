# Vue.js application project template for module type.
Vue.jsをモジュール版で始める時のテンプレートプロジェクト。
vue-cliではいろんなものが入るので、軽量版としてつかう。

## パッケージ
- babel-core
- babel-loader
- babel-preset-env
- css-loader
- html-webpack-plugin
- style-loader
- vue
- vue-loader
- vue-template-compiler
- webpack
- webpack-dev-server

## コマンド
- `npm run build` Vueのビルド    
- `npm run start` サーバ(webpack-dev-server)開始

## URL
`http://localhost:3000/`
